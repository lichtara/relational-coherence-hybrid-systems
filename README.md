# Toward a Structural Model of Relational Coherence for Hybrid Human–AI Systems

This repository contains the working materials for an academic paper proposing a
structural model of relational coherence applicable to hybrid human–AI–institutional systems.

## Core Contribution
The paper introduces a triadic structural model based on:
- Field
- Coherence
- Limit

as necessary conditions for systemic governability and sustainability.
